This project is when Interbyte Studios, the studio in which all the projects I've lead have been under, established an official development team. I scripted all front-end features and security - I also scripted the UI and responsive UX. This project, although is seems simple, has a lot of complex features to it, such as the third-party overseer application known as "Raptor", which is an overlord system that allows us to log all actions done within the game, and allow players to trade items between servers. This project is truly awesome!
